# Wanderlust Project

Wanderlust is a full stack web application inspired by Airbnb where users can create and explore travel listings.


## 🚀 Features

- Create new listings
- View all listings
- Edit existing listings
- Delete listings
- Image display for listings
- Responsive UI using Bootstrap

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap

## 📂 Project Structure

wanderlust-project
│
├── models
│   └── listing.js
├── views
│   └── listings
├── public
├── app.js
├── package.json


## ▶️ How to Run the Project

1. Clone the repository
2. Install dependencies

   npm install

3. Start the server

     nodemon app.js

4. Open browser

http://localhost:8080
